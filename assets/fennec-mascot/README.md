## 🎨 デザイン特徴

- **カラー**: クリーム色の毛並み、ピンクの耳の内側、ミントのスカーフ
- **スタイル**: ミニマルなかわいいスタイル（LINE Friends / Sanrio風）
- **コンセプト**: 大きな耳で「音を聞く」→ 音声日記アプリにぴったり
- **サイズ**: 1024x1024px（高解像度）
